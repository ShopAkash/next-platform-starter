## Why Choose Next.js?

Next.js provides several advantages over vanilla React:

1. **Server-Side Rendering (SSR)** - Better SEO and performance
2. **Static Site Generation (SSG)** - Lightning-fast loading times
3. **API Routes** - Build full-stack applications
4. **File-based Routing** - Intuitive page structure

## Getting Started

To create a new Next.js project, run:

```bash
npx create-next-app@latest my-app
```
